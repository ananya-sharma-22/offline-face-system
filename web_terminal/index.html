EDGE DEPLOYMENT constraints

Network: Strictly zero-dependency. All models packaged in app bundle. No API calls during inference.

Compute: Target 30+ FPS on consumer-grade x86/ARM devices without dedicated GPU.

Memory: Sub-500MB runtime footprint. FAISS index kept in memory for microsecond latency.

MODEL SPECIFICATIONS

BlazeFace (Detection) Lightweight detector optimized for mobile/edge. Outputs bounding box + 6 key facial landmarks. INT8 Quantized.

MobileFaceNet (Recognition) 1M parameter model producing 512-D embeddings. Trained on MS-Celeb-1M. INT8 Quantized.

MiniFASNet (Anti-Spoofing) Evaluates spatial textures for presentation attacks (printed photos, screens). Auxiliary model.
